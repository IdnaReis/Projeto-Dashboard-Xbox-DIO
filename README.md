# Dashboard de Vendas do Xbox

*Aluna: Idna da Silva Cunha dos Reis*

Projeto desenvolvido para criar um painel de visualização de dados (Dashboard) utilizando o Microsoft Excel, com foco na análise de vendas e assinaturas de produtos Xbox.

### 🎯 Objetivo
Transformar dados brutos em informações visuais claras e objetivas, permitindo uma análise eficaz do desempenho de vendas e auxiliando na tomada de decisões estratégicas.

### 🛠 Ferramentas e Recursos Utilizados
- Microsoft Excel 365
- Formatação Condicional
- Gráficos Dinâmicos
- Tabelas
- Identidade Visual e Paleta de Cores Oficial Xbox

### 📂 Conteúdo do Repositório
- dashboard_xbox_finalizado.xlsx: Arquivo completo com o dashboard desenvolvido, pronto para visualização e análise.
